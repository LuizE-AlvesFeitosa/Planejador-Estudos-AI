# Planejador-Estudos-AI
Aplicação web que cria planos de estudo personalizados com IA. Usuário informa disciplina, matérias e tempo disponível, e o sistema gera cronograma detalhado com tópicos diários e atividades. Backend em Python/Flask com API da Groq (llama-3.3-70b-versatile), frontend em HTML, CSS e JavaScript. Ferramenta prática para organização de estudos.
